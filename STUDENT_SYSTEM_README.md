# نظام إدارة الطلاب

تم إنشاء نظام شامل للطلاب يتيح لهم الوصول إلى مساحة شخصية تحتوي على:

## الميزات المُضافة

### 1. نظام تسجيل الدخول للطلاب
- صفحة تسجيل دخول مخصصة للطلاب (`/student-login`)
- دعم البريد الإلكتروني وكلمة المرور
- إعادة توجيه تلقائية إلى لوحة تحكم الطالب

### 2. مساحة الطالب الشخصية
- **لوحة التحكم**: إحصائيات سريعة وجلسات قادمة
- **الدروس**: عرض جميع الدروس المسجل بها الطالب
- **الحضور**: تتبع الحضور والغياب
- **الإشعارات**: نظام إشعارات تفاعلي
- **الملف الشخصي**: إدارة البيانات الشخصية

### 3. إنشاء الحسابات تلقائياً
- عند إضافة طالب جديد في لوحة الإدارة
- إذا لم يتم تحديد كلمة مرور، يتم إنشاؤها تلقائياً
- الصيغة: الحرف الأول من الاسم + 4 أرقام عشوائية
- عرض كلمة المرور المولدة للإداري

### 4. الواجهات الخلفية
- `StudentController`: إدارة جميع عمليات الطلاب
- `StudentService`: منطق العمل وإنشاء كلمات المرور
- `StudentRoutes`: تعريف المسارات الخلفية
- دعم JWT للمصادقة

### 5. الواجهات الأمامية
- `StudentLayout`: تخطيط موحد لجميع صفحات الطالب
- صفحات منفصلة لكل قسم
- تصميم متجاوب ومتوافق مع اللغة العربية
- تكامل مع نظام المصادقة

## كيفية الاستخدام

### للإداريين
1. انتقل إلى قسم "الطلاب" في لوحة الإدارة
2. اضغط "إضافة طالب جديد"
3. املأ البيانات المطلوبة
4. اترك حقل كلمة المرور فارغاً للتوليد التلقائي
5. احفظ وسيتم عرض كلمة المرور المولدة

### للطلاب
1. انتقل إلى `/student-login`
2. أدخل البريد الإلكتروني وكلمة المرور
3. ستتم إعادة التوجيه إلى مساحة الطالب
